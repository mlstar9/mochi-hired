# Pika Clean (DEFAULT STYLE)

> **This is the default style preset for all animation work.**
> Only use other styles (esports-punk, kda-cyberpop, street-minimal) when the brief specifically requires it.

## Brand Colors
```
purple:  #C2BEFF  (main accent, like Google's blue)
black:   #111111  (backgrounds)
yellow:  #FEFBCF  (highlights, warm accents)
beige:   #FDF7EF  (warm backgrounds, cards)
```

## Typography
- **Headings:** Telka-Extended (bold, extended width)
- **Body/Code:** SpaceMono (monospace)
- Keep text minimal - few words, big impact

## Animation Timing (Pika Style - inspired by Google/Gemini)

### Core Rules
- **HARD CUTS ONLY** - no fades, no ease in/out
- **NO slide/scale** unless explicitly requested
- **Clean and precise** - every frame intentional

### Timing
- **Hold beats:** Let important moments breathe (1-2s holds)
- **Text on screen:** 2+ seconds for readability
- **Quick cuts:** Don't linger, but don't rush

### Staggered Text (when used)
- Each line appears with hard cut
- Delay between lines: 0.5-1s
- NO fade, NO slide - just *appears*

### Pacing Guidelines
- **Scene duration:** 3-5 seconds minimum
- **Text readability:** 2+ seconds per text block
- **Transitions:** Hard cuts or VHS-style glitch (no crossfades)
- **Let it breathe:** Space between elements appearing

## Visual Patterns (Pika style)

### Backgrounds
- Pure black (#111111) for dramatic moments
- Beige (#FDF7EF) for warm, friendly feels
- Subtle gradients (black → purple tint)

### UI Mockups
- Rounded corners (12-16px)
- Soft shadows (purple-tinted)
- Floating cards with depth
- Zoomed-in details with cursor

### Decorative Elements
- Pika bunny logo
- Subtle purple glow/accent lighting
- Clean geometric shapes

## Style Preset: "Pika Clean"

**Use this preset for all Pika motion graphics unless stated otherwise.**

### Summary
Google/Gemini-inspired timing + Pika brand

### Colors
- Primary: `#C2BEFF` (purple)
- Background: `#111111` (black) or `#FDF7EF` (beige)
- Accent: `#FEFBCF` (yellow)
- Text: `#FFFFFF` on dark, `#111111` on light

### Typography
- Headings: Telka-Extended Black/Bold
- Body: Telka-Extended Medium or SpaceMono

### Animation
- **Cuts:** Hard only, no fades
- **Motion:** None unless requested
- **Stagger:** 0.5-1s between elements
- **Hold:** 2s minimum for text readability

### Transitions
- Hard cut (default)
- VHS glitch (for dramatic moments)
- NO crossfades, NO dissolves

### Composition
- Centered text on clean backgrounds
- One idea per screen
- Generous white/black space
- Pika logo for end cards
